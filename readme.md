# 学籍管理系统

## 1、项目介绍

学籍管理系统中可以对专业、班级、课程、学籍、成绩、奖惩进行管理


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.5-5.7、8.x版本都可） 
- 硬件环境：Windows 或者 Mac OS
